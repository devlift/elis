Course Editing Help

  • Name
    The name of the course.

  • ID Number
    A unique identifying value for this course (i.e. a course code like "CPS102").

  • Syllabus
    An explanation of the course or a summary of the topics covered in this course.

  • Length
    Enter a value here and specify the time duration (Days, Weeks, Months) for this value. NOTE: Leaving the value empty will not store a length with this course.

  • Environment
    If you have eny environment values defined, they will be listed here in a drop-down.
